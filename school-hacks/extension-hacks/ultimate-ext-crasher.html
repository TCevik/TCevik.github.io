Step 1: Identify the extension ID

Find the extension you want to disable and copy the 32-character ID of the extension.
You can find the ID in chrome://extensions and then the extension you want to crash in the url after chrome://extensions/?id=[ID IS HERE] or in chrome://extensions-internals.

Step 2: Open an extension page

Go to the URL chrome-extension://extension-id/manifest.json (replace "extension-id" with the actual ID of the extension).
This will open a page belonging to the extension you want to disable.

Step 3: Create bookmarks

Create three bookmarks: one for the extension page (bookmark A), one for chrome://kill (bookmark B), and one for chrome://hang (bookmark C).

Step 4: Disable the extension

While on the extension page (bookmark A), click on the chrome://kill bookmark (bookmark B). This should crash the page. Now quickly proceed to step 5.

Step 5: Act quickly

Immediately after the page crashes, start spamming the chrome://hang bookmark (bookmark C) and reload the page by clicking the refresh button.
You have to keep spamming bookmark C for about 5 seconds.
You need to reload the page within one or two seconds of the crash.

Step 6: Flex in class until it's fixed

You can now test if it has crashed by clicking bookmark A, it should keep loading indefinitely or click on the extension icon and nothing happens. You might need to try it a few times.
