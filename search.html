Install Extension

Follow these steps to install the TC_tam Search extension in Chrome:

  1. Download the extension: Click here to download the extension (.zip)
  2. Extract the zip file to a folder on your computer (e.g., in your Documents).
  3. Open a new tab and navigate to: chrome://extensions/
  4. In the top-right corner, enable Developer mode.
  5. In the top-left, click Load unpacked and select the extracted folder (the folder containing `manifest.json`).
