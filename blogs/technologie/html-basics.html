Een simpele website werkt met HTML, CSS en JavaScript. Hier ga je leren over een paar van de basis elementen van HTML, CSS en JavaScript.

HTML:

We zullen beginnen met HTML. Hiermee definieer je de structuur van je website.

  • <h1>, <h2>, <h3>, of <h4> - Titels kan je op een website zetten. Hoe groter het getal achter de h, hoe kleiner de titel.
  • <p> - Normale tekst kan je op een website zetten.
  • <a> - Een link kan je op een website zetten.
  • <button> - Een knop kan je op een website zetten.
  • <ul> - Een lijst kan je op een website zetten.
  • <li> - Een lijst item kan je op een website zetten.
  • <img> - Een afbeelding kan je op een website zetten.
  • <video> - Een video kan je op een website zetten.

Met deze elementen kan je een basispagina maken, maar er mist nog een hoop voor een functionele pagina. Daarvoor kan je attributen gebruiken. Die werken op de volgende manier:
Bijvoorbeeld bij de img tag moet je ook een foto definieren. Dat doe je zo: <img src='URL_VAN_FOTO'>. Dit zijn een paar attributen die je kan gebruiken en voor welk van de basiselementen die je kan gebruiken.
Je hebt ALTIJD een openende tag en een sluitende (<p></p>) tag bij elk soort element. Let daarbij goed op de /.

  • <a> (link):
    • href: De URL waar de link naartoe verwijst.
      Bijvoorbeeld: <a href="https://voorbeeld.com">Klik hier</a>
    • target: Bepaalt of de link in een nieuw tabblad wordt geopend.
      Bijvoorbeeld: <a href="https://voorbeeld.com" target="_blank">Klik hier</a>
  • <img> (afbeelding):
    • src: De URL van de afbeelding.
      Bijvoorbeeld: <img src="url_van_foto.jpg">
    • alt: Alternatieve tekst voor de afbeelding als deze niet geladen kan worden.
      Bijvoorbeeld: <img src="url_van_foto.jpg" alt="Beschrijving van de afbeelding">
    • width: De breedte van de afbeelding.
      Bijvoorbeeld: <img src="url_van_foto.jpg" width="300">
    • height: De hoogte van de afbeelding.
      Bijvoorbeeld: <img src="url_van_foto.jpg" height="200">
  • <video> (video):
    • src: De URL van de video.
      Bijvoorbeeld: <video src="video.mp4"></video>
    • controls: Geeft de videospeler bedieningsknoppen.
      Bijvoorbeeld: <video src="video.mp4" controls></video>
    • autoplay: Laat de video automatisch afspelen.
      Bijvoorbeeld: <video src="video.mp4" autoplay></video>
    • loop: Laat de video herhalen.
      Bijvoorbeeld: <video src="video.mp4" loop></video>
  • <button> (knop):
    • type: Bepaalt het type van de knop (bijvoorbeeld "submit" voor een verzendknop in een formulier).
      Bijvoorbeeld: <button type="submit">Verzend</button>
    • onclick: Voert een JavaScript-functie uit bij klikken.
      Bijvoorbeeld: <button onclick="mijnFunctie()">Klik hier</button>

Nu, voordat we verder gaan naar de volgende talen om een website te creëeren gaan we eerst leren hoe je een link naar een ander bestand kan maken. Dat doen we zodat we een extern CSS (layout) en JavaScript (interactiviteit) bestand kunnen linken om onze website beter te maken.

Dat werkt bij een JavaScript en CSS bestand allebei anders.
Bij een .js (JavaScript) bestand gaat dat zo: <script src="url_van_js_bestand.js"></script>
Bij een .css bestand gaat dat zo: <link rel="stylesheet" href="url_van_css_bestand.css"></link>

CSS:

CSS wordt gebruikt om de pagina aan te passen qua layout en kleuren. Dit is essentieel voor een website.

Je hebt een paar belangrijke aspecten van de taal CSS nodig om een goede website te maken:

  • <margin> - Dit is de ruimte om een element heen.
  • <padding> - Dit is de ruimte binnenin een element, tussen de inhoud en de rand.
  • <border> - Dit is de rand die een element omringt en kan verschillende stijlen, diktes en kleuren hebben.
  • <background-color> - Hiermee kun je de achtergrondkleur van een element instellen.
  • <font-size> - Dit bepaalt de grootte van de tekst in een element.
  • <color> - Hiermee stel je de kleur van de tekst in.
  • <display> - Dit bepaalt hoe een element wordt weergegeven (bijvoorbeeld inline, block, flex, grid).
  • <width> en <height> - Hiermee kun je de breedte en hoogte van een element instellen.
  • <flexbox> - Een krachtige layouttechniek die je helpt om flexibele en responsieve layouts te creëren.
  • <grid> - Een andere layouttechniek die je in staat stelt om complexere layouts te maken met rijen en kolommen.
  • <media queries> - Hiermee kun je verschillende stijlen toepassen afhankelijk van het schermformaat of apparaatspecificaties, wat essentieel is voor responsive design.
  • <position> - Hiermee kun je de positie van een element op de pagina bepalen (bijvoorbeeld static, relative, absolute, fixed).
  • <overflow> - Dit bepaalt hoe inhoud die buiten de container valt, wordt behandeld (bijvoorbeeld hidden, scroll, auto).
  • <opacity> - Hiermee kun je de transparantie van een element instellen.
  • <text-align> - Hiermee kun je de uitlijning van tekst in een element instellen (bijvoorbeeld left, right, center, justify).
  • <line-height> - Dit bepaalt de hoogte van de regel voor tekst, wat invloed heeft op de leesbaarheid.
  • <font-family> - Hiermee kun je het lettertype van de tekst instellen.
  • <cursor> - Hiermee kun je het type cursor instellen dat verschijnt wanneer de muis over een element beweegt (bijvoorbeeld pointer, text, wait).
  • <transition> - Hiermee kun je animaties en overgangen tussen verschillende stijlen instellen.
  • <animation> - Hiermee kun je complexere animaties aan elementen toevoegen.
  • <box-shadow> - Dit voegt schaduwen toe aan elementen, wat diepte kan toevoegen aan je design.
  • <text-shadow> - Hiermee kun je schaduwen aan tekst toevoegen voor een visueel effect.

Css kan je defineren met <style>, dat ziet er zo uit:

                <style>
                button {
                margin: 10px;
                padding: 5px;
                font-family: Arial;
                color: lime;
                opacity: 0.5;
                }
                </style>

Dit zal alle <button> elementen beïnvloeden.

JavaScript:

JavaScript is een programmeertaal die je kunt gebruiken om je website interactief te maken. Het stelt je in staat om dingen te doen zoals reageren op klikken, formulieren verzenden, of dynamisch inhoud wijzigen zonder de pagina opnieuw te laden. Dit is essentieel voor een moderne webervaring.

Hier zijn een paar belangrijke concepten van JavaScript die je moet weten:

  • Variabelen - Dit zijn containers waarmee je gegevens kunt opslaan. Je declareert een variabele met let, const of var. Bijvoorbeeld: 
    let naam = "Tamer";
  • Functies - Dit zijn blokken code die je kunt hergebruiken. Je kunt een functie definiëren met function: 
    function zegHallo() { alert("Hallo, " + naam + "!"); }
  • Events - Dit zijn acties die plaatsvinden in de browser, zoals klikken op een knop of het bewegen van de muis. Je kunt deze gebeurtenissen gebruiken om functies uit te voeren. Bijvoorbeeld: 
    <button onclick="zegHallo()">Klik hier</button>
  • Conditionele statements - Hiermee kun je beslissingen nemen in je code. Een voorbeeld is de if statement: 
    if (naam === "Tamer") { console.log("Welkom terug, Tamer!"); }
  • Lussen - Dit zijn manieren om herhaalde taken uit te voeren. Een voorbeeld is de for lus: 
    for (let i = 0; i < 5; i++) { console.log("Dit is regel " + i); }
  • Arrays - Dit zijn lijsten van waarden. Je kunt arrays gebruiken om meerdere waarden op te slaan in één variabele. Bijvoorbeeld: 
    let kleuren = ["rood", "groen", "blauw"];
  • Objecten - Dit zijn complexere datastructuren die waarden in paren van sleutel-waarde opslaan. Bijvoorbeeld: 
    let persoon = { naam: "Tamer", leeftijd: 15 };

Je kunt JavaScript aan je HTML-bestand toevoegen met de <script> tag. Dit kan direct in je HTML, of via een extern bestand:

<script src="url_van_js_bestand.js"></script>

Hier is een eenvoudig voorbeeld dat alles samenbrengt:

<button onclick="zegHallo()">Klik hier</button>

Met deze basis van JavaScript kun je beginnen met het toevoegen van interactie aan je website en je site interactiever maken!
